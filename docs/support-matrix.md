# TypeWhisper 支持矩阵

本矩阵描述了直链下载版本官方支持的 `1.0` 路线。

## 平台

| 领域 | 支持情况 |
| --- | --- |
| 基础支持 | macOS 14+ |
| 推荐硬件 | Apple Silicon |
| Intel | 只要宣传了通用二进制支持，每次发布前进行冒烟测试 |

## 按 macOS 版本划分的功能矩阵

| 功能 | macOS 14 | macOS 15 | macOS 26+ | 备注 |
| --- | --- | --- | --- | --- |
| 系统级听写 | 是 | 是 | 是 | `1.0` 核心工作流 |
| 文件转录 | 是 | 是 | 是 | `1.0` 核心工作流 |
| 提示词处理 | 是 | 是 | 是 | `1.0` 核心工作流 |
| Profile、历史、词典、代码片段 | 是 | 是 | 是 | `1.0` 核心工作流 |
| Widgets | 是 | 是 | 是 | 非核心路径 |
| HTTP API | 是 | 是 | 是 | 仅限回环地址，默认禁用 |
| CLI | 是 | 是 | 是 | 需要本地 API 服务器运行 |
| Apple Translate 集成 | 否 | 是 | 是 | 高级功能面 |
| 改进的设置界面 | 否 | 是 | 是 | 可选的可用性改进 |
| Apple Intelligence 提供商 | 否 | 否 | 是 | 可选，非核心路径 |
| SpeechAnalyzer 引擎 | 否 | 否 | 是 | 可选，非核心路径 |

## 引擎说明

| 引擎类型 | 1.0 支持情况 | 备注 |
| --- | --- | --- |
| 本地引擎 | 是 | 推荐默认路径 |
| 云端引擎 | 是 | 需要有效 API 密钥 |
| 捆绑插件 | 是 | 已测试产品路径的一部分 |
| 外部第三方插件 | 尽力而为 | 非 `1.0` 发布阻塞项 |
